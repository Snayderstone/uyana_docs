# Funcionalidades clave

Las herramientas principales de Uyana para el día a día.

## Gestión de proyectos

- Crea un proyecto desde **Nuevo > Proyecto**.
- Define responsables y fechas; las tareas heredarán esta configuración por defecto.
- Usa etiquetas para agrupar iniciativas y facilitar la búsqueda.

## Tareas y seguimiento

- Crea tareas desde el tablero o el panel de proyecto.
- Cambia de estado arrastrando en el tablero Kanban.
- Agrega comentarios y menciona a tu equipo con `@usuario`.
- Adjunta archivos o enlaces; se guardan en el historial de la tarea.

## Reportes y métricas

- El módulo **Reportes** ofrece vistas predefinidas (progreso, capacidad, riesgos).
- Usa filtros por fechas, etiquetas o responsable.
- Exporta a CSV o PDF para compartir con stakeholders.

## Roles y permisos

- Administradores: gestionan usuarios, permisos y plantillas.
- Colaboradores: crean y editan tareas y proyectos.
- Lectores: acceso de solo lectura a proyectos asignados.
